# kpi-management-system

## Tech Stack
React.js + Express.js + MongoDB + DigitalOcean

## Setup instructions
```
docker-compose up -d   // for run all services

docker-compose ps     // list container
```

```
frontend: http://localhost:3001
backend : http://localhost:3000/health
```


## API documentation
```
cd backend  (folder)

// api doc file
api-auth.http
api-dashbaord.http
api-kpi.http
api-user.http
```
